# Terminal Trade Bot 🤖

Экономическая игра-трейдер в Telegram

## 🎮 Как играть
- Покупай и продавай акции
- Зарабатывай деньги
- Покупай предметы в магазине
- Соревнуйся с другими игроками

## 📋 Команды
- `/start` - начать игру
- `/tbuy AAPL 5` - купить 5 акций Apple
- `/tsell AAPL 5` - продать 5 акций Apple
- `/tnext` - следующий день
- `/tstats` - твоя статистика
- `/tlist` - список компаний
- `/tleader` - таблица лидеров
- `/tpromo КОД` - активировать промокод
- `/shop` - магазин
- `/inventory` - инвентарь

## 🚀 Установка на BotHost
1. Загрузи все файлы на GitHub
2. На BotHost укажи ссылку на репозиторий
3. Укажи команду запуска: `python bot.py`
4. Готово!

## 🛠 Для разработки
- Python 3.8+
- SQLite (встроен)
- pyTelegramBotAPI
